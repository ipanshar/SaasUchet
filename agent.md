# agent.md

Совместимый файл для репозиториев, где ранее использовалось имя `agent.md`.

Основной стандартный файл инструкций агента теперь: `AGENTS.md`.

Смотреть в первую очередь:

1. `AGENTS.md` — правила работы агента и завершения сессии.
2. `CLAUDE.md` — архитектура проекта, стек и технические соглашения.
